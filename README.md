## Install

> All instructions can be found at [draculatheme.com/hyperterm](https://draculatheme.com/hyperterm).

Simply add `hyperterm-dracula` to the plugins list in your `~/.hyperterm.js` config file:

```js
module.exports = {

  config: { /*... */ },

  plugins: [
    "hyperterm-dracula"
  ]

};
```
